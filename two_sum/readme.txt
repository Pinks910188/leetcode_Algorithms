##心得  

#java  

java 有一個很特別的用法，`hashmap`，解題的時候，當然一開始直覺就是兩個迴圈輕而易舉的就得出答案，但是速度上一定是偏慢的(畢竟兩個迴圈)  

而要是這種查找已知的element在陣列裡，hashmap 可以使O(n^2)->O(n) 這裡有個不錯的網站，https://www.itread01.com/articles/1478501406.html

#python  

python似乎沒有java_hashmap的用法，python的map是帶入陣列與function去做陣列處理的用法，所以我是用暴力解，有機會可以嘗試用python自己寫一個hashmap的功能  

#C++  

C++有類似java hashmap的用法，所以解法與java很像。
